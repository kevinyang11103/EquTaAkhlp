# Python图书管理系统 python272

## 项目概述

本项目是一个基于Python开发的图书管理系统，包含了源码、运行教程、开发文档以及PPT介绍，旨在为用户提供一个结构清晰、易于理解的图书管理解决方案。

## 技术栈

- 后端：Django
- 数据库：MySQL
- 前端：HTML + CSS + JavaScript

## 功能模块

- **用户系统**
  - 登录
  - 注册
- **图书信息管理**
  - 添加
  - 修改
  - 查询
  - 删除
- **图书借阅记录**
  - 借阅记录管理
- **读者信息查询**
  - 读者信息管理

## 系统角色

- 系统管理员
- 图书管理员
- 读者

## 运行环境

- Python环境
- Django框架
- MySQL数据库

## 部署步骤

1. 配置Python环境。
2. 安装Django框架。
3. 配置MySQL数据库。
4. 根据开发文档和运行教程，进行系统部署。
